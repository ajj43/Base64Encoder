# base64Encoder lib


## Overview

libtool, curl is required to building this project.

brew install autoconf automake libtool
brew install curl

## Execute

- `./autogen.sh` 
- `./configure`
- `make`

## Commandline

- `./demo IMAGE_URL OUTPUT_FILE_PATH`

ex: `/demo https://example.com ~/Desktop/baseencoded.txt`
